# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful and constructive
- Giving actionable feedback
- Assuming good intent
- Welcoming contributors at all experience levels

Examples of unacceptable behavior:

- Harassment, discrimination, or personal attacks
- Trolling or intentionally disruptive behavior
- Publishing others' private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, or pull requests that violate this Code of Conduct.

## Reporting

Please report issues to the repository owner through GitHub private contact methods when available.
