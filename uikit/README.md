# agent-uikit Android library

Reusable Jetpack Compose components for conversational AI, voice, video, and biomarker-oriented Android interfaces.

This module is the publishable library inside the repository. The `app` module is only a showcase consumer.

## Build

```bash
export JAVA_HOME="/Applications/Android Studio.app/Contents/jbr/Contents/Home"
./gradlew :uikit:assemble
```
